快速捕获一个新任务到工作流库。

## 用法

```
/capture <任务描述> [优先级] [截止日期]
```

## 示例

```
/capture 修复登录页面的 bug
/capture 完成需求文档 高优 2026-03-25
/capture 和设计师对齐 UI 方案 紧急
```

## 行为

1. 解析用户输入，提取任务名称、优先级（可选）、截止日期（可选）
2. 调用 `create_task` 创建任务，状态默认为"待办"
3. 输出创建成功的任务信息，包含任务 ID 和 Notion 链接

## 优先级关键词识别

- 紧急 / urgent / ASAP → 🔴 紧急
- 高优 / 高 / high → 🟡 高
- 普通 / 低 / normal（默认）→ 🟢 普通
