生成本周工作复盘报告。

## 用法

```
/review          # 本周复盘
/review last     # 上周复盘
```

## 行为

1. 调用 `generate_weekly_review(week_offset)` 获取数据
2. 以复盘报告格式输出：

```
📅 2026-W12 工作复盘

📊 数据概览
  ✅ 完成任务：5 项
  🔄 进行中：2 项
  📝 新增笔记：8 篇

✅ 本周完成
  • [🔴] 修复登录 Bug（2026-03-18）
  • [🟡] 完成需求文档（2026-03-19）
  • [🟢] 更新依赖库（2026-03-20）
  ...

🔄 仍在进行
  • 重构用户模块
  • 性能优化调研

💡 下周重点
  （Claude 根据进行中任务自动建议）
```

3. 末尾可附上 Claude 的简短分析和下周建议
