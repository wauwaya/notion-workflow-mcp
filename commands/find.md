在工作流库和笔记库中同时搜索关键词。

## 用法

```
/find <关键词>
```

## 示例

```
/find 登录
/find redis 缓存
```

## 行为

1. 并行调用 `search_tasks(query)` 和 `search_notes(query)`
2. 合并结果，分组展示：

```
🔍 搜索 "登录" 的结果

📋 任务（2 个）
  • [进行中] 修复登录 Bug  ID: abc123
  • [完成]   登录页重构    ID: def456

📝 笔记（3 个）
  • 登录接口并发问题分析   2026-03-20
  • 登录流程设计文档       2026-03-18
  • OAuth 登录参考资料     2026-03-15
```

3. 如果无结果，提示"未找到相关内容"
