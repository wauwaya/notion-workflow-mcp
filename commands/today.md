查看今日相关任务：今天到期的任务 + 所有进行中的任务。

## 用法

```
/today
```

## 行为

1. 调用 `get_today_tasks` 获取今日任务列表
2. 按以下格式输出：

```
📅 今日任务 (2026-03-21)

🔄 进行中
  • [🔴] 修复登录 Bug  (due: 今天)  ID: xxx
  • [🟡] 完成需求文档              ID: xxx

📋 今日到期
  • [🟢] 周报整理                  ID: xxx
```

3. 如果没有任务，提示"今日暂无任务，使用 /capture 添加新任务"
