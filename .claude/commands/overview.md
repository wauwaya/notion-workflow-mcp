查看工作流库的任务状态总览仪表盘。

## 用法

```
/overview
```

## 行为

1. 调用 `get_overview` 获取各状态任务数量
2. 以仪表盘格式展示：

```
📊 工作流概览

  📋 待办      ████░░░░░░  8 项
  🔄 进行中    ██░░░░░░░░  3 项
  ✅ 完成      ██████████  21 项
  ⏸  搁置      █░░░░░░░░░  2 项

  合计：34 项任务
```

3. 附上快捷提示：
   - 有"进行中"任务时：提示可用 `/review today` 查看详情
   - 无"进行中"任务时：提示可用 `/capture` 开始新任务
