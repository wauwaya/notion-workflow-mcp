汇总当前对话中的知识要点，保存为笔记。

## 用法

/note

## 行为

1. AI 回顾当前对话中最近的问答内容（特别是 AI 解答的知识性内容）
2. 汇总为结构化笔记：
   - 自动生成标题（简明概括主题）
   - 正文为 Markdown 格式，包含关键要点、代码示例等
3. 展示汇总预览给用户确认
4. 询问用户选择标签：
   - 列出笔记库常用标签作为候选（如有历史数据）
   - 支持用户自定义输入新标签
5. 调用 MCP `create_note(title, content, note_type="速记", tags)` 保存
6. 输出确认：

📝 笔记已保存「{标题}」
   标签: {tag1}, {tag2}
   🔗 {notion_url}
