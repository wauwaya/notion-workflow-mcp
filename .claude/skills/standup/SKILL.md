生成每日站会内容（昨日完成 / 今日计划 / 阻塞项）。

## 用法

```
/standup
```

## 行为

1. 调用 `generate_standup` 获取报告数据
2. 以标准站会格式输出：

```
📋 每日站会 — 2026-03-21

✅ 昨日完成
  • 修复登录 Bug
  • 完成需求文档 v1

🎯 今日计划
  • 代码 Review
  • 和设计师对齐 UI 方案

⚠️ 阻塞 / 风险
  • 等待后端接口联调（依赖小王）
```

3. 如果某个分类为空，显示"暂无"
4. 可以选择复制到剪贴板（如果环境支持）
